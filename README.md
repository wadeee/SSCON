# SSCON

Shadowsocks service management client

## Steps

+ System

    CentOS 7

+ Environment

    git, nginx and python installed

+ Download the source

    ```shell script
    git clone https://github.com/wadeee/sscon.git ~/sscon
    cd ~/sscon
    ```

+ Run inst.sh

    ```shell script
    ./inst.sh
    ```

+ Check the page

    ```text
    http://localhost/
    ```


## Tech

+ [Python](https://www.python.org/)
